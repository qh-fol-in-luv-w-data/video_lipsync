# Job Description: Junior PR Executive

**Phòng ban / Lĩnh vực:** Marketing & Truyền Thông (Marketing & PR)

## 1. Mô Tả Công Việc
- Viết thông cáo báo chí, liên hệ booking báo/KOL, hỗ trợ tổ chức sự kiện.

## 2. Kỹ Năng Cứng (Hard Skills)
- Media Relations, PR writing, Event coordination.

## 3. Kỹ Năng Mềm (Soft Skills)
- Giao tiếp khéo léo, xử lý tình huống, ngoại giao.

## 4. Công Cụ & Phần Mềm (Tools)
- MS Office, Email Marketing tools.
