# Job Description: Junior DevOps Engineer

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Hỗ trợ quản lý server, cấu hình CI/CD pipeline, monitor hệ thống.

## 2. Kỹ Năng Cứng (Hard Skills)
- Linux cơ bản, Bash script, Docker, Networking cơ bản.

## 3. Kỹ Năng Mềm (Soft Skills)
- Phản ứng nhanh, cẩn thận, học hỏi liên tục.

## 4. Công Cụ & Phần Mềm (Tools)
- Linux, Docker, Jenkins/GitLab CI, AWS cơ bản.
