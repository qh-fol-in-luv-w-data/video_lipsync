# Job Description: Junior Mobile Developer (iOS/Android)

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Phát triển app di động, thiết kế UI/UX trên mobile, kết nối API.

## 2. Kỹ Năng Cứng (Hard Skills)
- Swift/Kotlin hoặc Flutter/React Native, Mobile UI, API.

## 3. Kỹ Năng Mềm (Soft Skills)
- Sáng tạo, tỉ mỉ, khả năng debug tốt.

## 4. Công Cụ & Phần Mềm (Tools)
- Xcode/Android Studio, Git, Figma.
