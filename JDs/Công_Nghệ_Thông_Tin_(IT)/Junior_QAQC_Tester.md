# Job Description: Junior QA/QC Tester

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Viết test case, thực hiện manual test, log bug, theo dõi tiến độ fix bug.

## 2. Kỹ Năng Cứng (Hard Skills)
- Software Testing Life Cycle, viết Test Case, SQL cơ bản.

## 3. Kỹ Năng Mềm (Soft Skills)
- Cực kỳ cẩn thận, giao tiếp tốt với Dev.

## 4. Công Cụ & Phần Mềm (Tools)
- Jira, Trello, Postman, Excel.
