# Job Description: Junior Data Analyst

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Thu thập, làm sạch dữ liệu, xây dựng dashboard, báo cáo phân tích cơ bản.

## 2. Kỹ Năng Cứng (Hard Skills)
- SQL, Python (Pandas), Data Visualization, Thống kê.

## 3. Kỹ Năng Mềm (Soft Skills)
- Tư duy phân tích, nhạy bén với số liệu.

## 4. Công Cụ & Phần Mềm (Tools)
- Excel, Power BI / Tableau, Jupyter Notebook.
