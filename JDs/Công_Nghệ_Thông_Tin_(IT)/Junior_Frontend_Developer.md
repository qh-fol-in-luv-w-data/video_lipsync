# Job Description: Junior Frontend Developer

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Phát triển giao diện web, cắt HTML/CSS từ Figma, tích hợp API, fix bug UI.

## 2. Kỹ Năng Cứng (Hard Skills)
- HTML/CSS, JavaScript (React/Vue/Angular), Responsive Design.

## 3. Kỹ Năng Mềm (Soft Skills)
- Tỉ mỉ, làm việc nhóm, tư duy giải quyết vấn đề.

## 4. Công Cụ & Phần Mềm (Tools)
- VS Code, Git, Figma, Chrome DevTools.
