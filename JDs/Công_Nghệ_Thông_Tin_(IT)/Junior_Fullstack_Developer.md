# Job Description: Junior Fullstack Developer

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Phát triển cả frontend và backend cho các tính năng nhỏ, hỗ trợ deploy.

## 2. Kỹ Năng Cứng (Hard Skills)
- JS/TS, Node.js, React/Vue, CSDL, Git.

## 3. Kỹ Năng Mềm (Soft Skills)
- Quản lý thời gian, khả năng xử lý vấn đề độc lập.

## 4. Công Cụ & Phần Mềm (Tools)
- VS Code, Git, Docker, Postman.
