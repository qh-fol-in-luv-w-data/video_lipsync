# Job Description: Junior System Admin

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Cài đặt phần cứng/phần mềm, hỗ trợ IT helpdesk, quản lý mạng nội bộ.

## 2. Kỹ Năng Cứng (Hard Skills)
- Windows Server/Linux, Networking (LAN/WAN), Troubleshooting.

## 3. Kỹ Năng Mềm (Soft Skills)
- Nhiệt tình, hỗ trợ khách hàng, giải quyết vấn đề.

## 4. Công Cụ & Phần Mềm (Tools)
- Active Directory, TeamViewer/AnyDesk, Cisco cơ bản.
