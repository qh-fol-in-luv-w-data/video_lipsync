# Job Description: Junior Data Engineer

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Xây dựng data pipeline cơ bản, hỗ trợ duy trì data warehouse.

## 2. Kỹ Năng Cứng (Hard Skills)
- SQL, Python, ETL concepts, Database management.

## 3. Kỹ Năng Mềm (Soft Skills)
- Cẩn thận, tư duy hệ thống.

## 4. Công Cụ & Phần Mềm (Tools)
- Airflow cơ bản, SQL Server/Postgres, AWS/GCP cơ bản.
