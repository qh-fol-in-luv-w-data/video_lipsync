# Job Description: Junior Backend Developer

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Xây dựng RESTful API, thiết kế database cơ bản, viết unit test, tối ưu code.

## 2. Kỹ Năng Cứng (Hard Skills)
- NodeJS/Python/Java/PHP, SQL/NoSQL, OOP, API Design.

## 3. Kỹ Năng Mềm (Soft Skills)
- Tư duy logic, kiên nhẫn, học hỏi nhanh.

## 4. Công Cụ & Phần Mềm (Tools)
- Postman, Git, Docker cơ bản, DBeaver.
