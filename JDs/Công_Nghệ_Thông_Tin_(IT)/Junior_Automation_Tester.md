# Job Description: Junior Automation Tester

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Viết script tự động hóa test web/app, tích hợp vào CI/CD cơ bản.

## 2. Kỹ Năng Cứng (Hard Skills)
- Java/Python, Selenium/Cypress, OOP cơ bản.

## 3. Kỹ Năng Mềm (Soft Skills)
- Tư duy logic, kiên nhẫn.

## 4. Công Cụ & Phần Mềm (Tools)
- Selenium, Cypress, Git, Jira.
