# Job Description: Junior AI/ML Engineer

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Chuẩn bị data, train các model học máy cơ bản, đánh giá model.

## 2. Kỹ Năng Cứng (Hard Skills)
- Python, Machine Learning algorithms, Math/Stats.

## 3. Kỹ Năng Mềm (Soft Skills)
- Nghiên cứu, kiên trì, phân tích.

## 4. Công Cụ & Phần Mềm (Tools)
- Scikit-learn, TensorFlow/PyTorch (cơ bản), Jupyter.
