# Job Description: Junior Cloud Engineer

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Hỗ trợ triển khai dịch vụ trên cloud, monitor chi phí và hiệu suất.

## 2. Kỹ Năng Cứng (Hard Skills)
- AWS/Azure/GCP cơ bản, Networking, IAM.

## 3. Kỹ Năng Mềm (Soft Skills)
- Tư duy hệ thống, cẩn thận.

## 4. Công Cụ & Phần Mềm (Tools)
- AWS Console, Terraform (cơ bản), Linux.
