# Job Description: Junior Cyber Security Analyst

**Phòng ban / Lĩnh vực:** Công Nghệ Thông Tin (IT)

## 1. Mô Tả Công Việc
- Monitor hệ thống cảnh báo bảo mật, phân tích log, dò quét lỗ hổng cơ bản.

## 2. Kỹ Năng Cứng (Hard Skills)
- Networking protocols, OS security, Vulnerability scanning.

## 3. Kỹ Năng Mềm (Soft Skills)
- Đạo đức nghề nghiệp, cẩn thận, phản xạ nhanh.

## 4. Công Cụ & Phần Mềm (Tools)
- Wireshark, Nmap, SIEM cơ bản.
